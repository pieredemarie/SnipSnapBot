# SnipSnapBot

Телеграм-бот для сохранения и управления ссылками с тегами.

## Функционал

- `/save <URL> <теги>` — сохранить ссылку с тегами
- `/list` — показать все ссылки пользователя
- `/list <тег>` — показать ссылки по конкретному тегу
- `/get` — получить случайную ссылку
- `/remove <URL>` — удалить ссылку
- `/edit <старый URL> <новый URL|теги>` — редактировать ссылку
- Любое другое сообщение — бот отвечает «Неизвестная команда»
